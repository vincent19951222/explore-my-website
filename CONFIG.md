# 🏝️ 落地页配置指南

本文档说明如何配置和自定义你的 3D 群岛落地页。

---

## 📍 配置位置

所有配置都在 `index.html` 文件中的 **`projects` 数组**（约第 310 行）。

---

## 🏗️ 岛屿地标类型

共支持 **5 种**地标类型：

| type | 外观 | 动画效果 |
|------|------|----------|
| `lighthouse` | 🗼 红白条纹灯塔 | 顶部发光 |
| `palm` | 🌴 弯曲棕榈树 | 6片下垂树叶 |
| `cabin` | 🏠 小木屋 | 门+烟囱+棱锥屋顶 |
| `windmill` | 🌀 风车 | 叶片持续旋转 |
| `ruins` | 🏛️ 废墟石柱 | 紫色水晶漂浮+旋转 |

---

## 📝 岛屿配置字段

```javascript
{
    x: 0,           // X 轴位置（正数向右，负数向左）
    z: 0,           // Z 轴位置（正数向后，负数向前）
    type: 'lighthouse',  // 地标类型（见上表）
    name: "项目名称",     // 显示在对话框标题
    description: "项目描述，支持较长文本，会以打字机效果显示。",
    cover: "https://xxx.png",  // 封面图 URL（150x150 推荐）
    link: "https://your-demo.com",  // 点击 [OPEN DEMO] 跳转地址
    themeColor: '#FF6B6B',  // 信息牌主题颜色（十六进制）
    ready: true  // ✅ 是否已配置完成
}
```

---

## ✅ 配置状态区分

| ready 值 | 信息牌 | 对话框按钮 | 描述文字 |
|----------|--------|------------|----------|
| `true` | 彩色 + 项目名 | `[ OPEN DEMO ] ▶` 可点击 | 正常显示 |
| `false` | 灰色 + "COMING SOON" | `[ COMING SOON ]` 禁用 | 前缀 🚧 + 后缀 "(即将上线)" |

---

## ➕ 新增岛屿

在 `projects` 数组末尾添加新对象即可：

```javascript
const projects = [
    // ... 现有岛屿
    {
        x: 20, z: 0, type: 'ruins',
        name: "新项目",
        description: "项目描述...",
        cover: "https://placehold.co/150x150/9B59B6/FFFFFF/png?text=New",
        link: "https://example.com",
        themeColor: '#9B59B6',
        ready: false  // ⏳ 未完成配置
    }
];
```

> **注意**：桥梁会自动从中心岛（第一个）连接到其他所有岛屿。

## 📐 位置布局建议

当前布局采用 **星形结构**（中心岛 + 周围分布）：

```
        (-14, 8)          (14, 6)
              \           /
               \         /
        (-10, -12) — (0, 0) — (16, -8)
                     |
                  (5, -14)
```

- **中心岛**：建议放最重要的项目，设为 `(0, 0)`
- **周围岛**：X/Z 绝对值建议在 `10-18` 范围
- **避免重叠**：相邻岛屿间距至少保持 `8` 单位

## 🔧 高级配置

### 调整相机初始位置

```javascript
const initialCamPos = { x: 0, y: 40, z: 70 };
```

- `y`: 高度（越大俯视角度越陡）
- `z`: 距离（越大看得越远）

### 调整岛屿悬浮幅度

在 `animate()` 函数中找到：

```javascript
const floatY = Math.sin(...) * 0.4;  // 0.4 为悬浮幅度
```
